Computational methods for inverse problems governed by PDEs

by Alen Alexanderian

About the Book

This book covers computational methods for solving inverse problems governed by partial differential equations (PDEs). Mathematical models governed by PDEs play an essential role in forecasting complex phenomena such geological flows, ocean dynamics, or weather systems. Such models typically include input parameters that are unknown/uncertain and need to be estimated. The predictive fidelity of mathematical models depend critically on accurate specification of these model parameters. Therefore, estimating uncertain model parameters is a crucial aspect of model-based predictions. This can be done by solving an inverse problem where one estimates model parameters of interest using measurement data and the mathematical model. We focus on inverse problems constrained by PDEs that have infinite-dimensional inversion parameters. Such problems are common in applications. In such inverse problems, the goal is to estimate function-valued parameters appearing, for example, as coefficient functions, source terms, or initial conditions. The book considers both deterministic and Bayesian formulations, with emphasis on computational methods.
